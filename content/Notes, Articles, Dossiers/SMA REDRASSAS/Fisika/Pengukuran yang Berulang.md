# Pengukuran Berulang

Adalah metode untuk memperoleh hasil pengukuran yang lebih teliti dengan cara mengukur suatu besaran secara berulang kali menggunakan alat ukur yang sama, kemudian menghitung nilai rata-rata dan ketidakpastian pengukurannya.

Metode ini dilakukan untuk mengurangi kesalahan acak dan mendekati nilai sebenarnya.

---

## Langkah-langkah Pengukuran Berulang

1. Lakukan pengukuran terhadap suatu besaran sebanyak ( n ) kali.
    
2. Catat semua hasil pengukuran ( x_1, x_2, x_3, \ldots, x_n ).
    
3. Hitung nilai rata-rata pengukuran.
    
4. Hitung simpangan atau ketidakpastian pengukuran.
    

---

### Rumus Pengukuran Berulang

1. **Nilai rata-rata pengukuran**
    

$$  
\bar{x} = \frac{\sum_{i=1}^{n} x_i}{n}  
$$

Keterangan:

- ( \bar{x} ) = nilai rata-rata
    
- ( x_i ) = hasil pengukuran ke-( i )
    
- ( n ) = banyaknya pengukuran
    

---

2. **Simpangan tiap data terhadap rata-rata**
    

$$  
\Delta x_i = | x_i - \bar{x} |  
$$

Keterangan:

- ( \Delta x_i ) = simpangan pengukuran ke-( i )
    
- ( x_i ) = hasil pengukuran ke-( i )
    
- ( \bar{x} ) = nilai rata-rata pengukuran
    

---

3. **Ketidakpastian rata-rata (simpangan rata-rata)**
    

$$  
\Delta \bar{x} = \frac{\sum_{i=1}^{n} \Delta x_i}{n}  
$$

Keterangan:

- ( \Delta \bar{x} ) = ketidakpastian rata-rata
    
- ( \Delta x_i ) = simpangan pengukuran ke-( i )
    
- ( n ) = banyaknya pengukuran
    

---

4. **Hasil akhir pengukuran**
    

$$  
X = \bar{x} \pm \Delta \bar{x}  
$$

Keterangan:

- ( X ) = hasil akhir pengukuran
    
- ( \bar{x} ) = nilai rata-rata pengukuran
    
- ( \Delta \bar{x} ) = ketidakpastian rata-rata
    

---

### Contoh Perhitungan

Misalkan panjang suatu benda diukur sebanyak 3 kali dengan hasil:  
( x_1 = 10{,}2 ) cm, ( x_2 = 10{,}3 ) cm, ( x_3 = 10{,}1 ) cm.

Nilai rata-rata:

$$  
\bar{x} = \frac{10{,}2 + 10{,}3 + 10{,}1}{3} = 10{,}2 \ \text{cm}  
$$

Simpangan:

$$  
\Delta x_1 = 0{,}0 \ \text{cm}, \quad \Delta x_2 = 0{,}1 \ \text{cm}, \quad \Delta x_3 = 0{,}1 \ \text{cm}  
$$

Ketidakpastian:

$$  
\Delta \bar{x} = \frac{0{,}0 + 0{,}1 + 0{,}1}{3} = 0{,}067 \ \text{cm}  
$$

Hasil akhir pengukuran:

$$  
X = (10{,}2 \pm 0{,}067) \ \text{cm}  
$$

---

Dengan pengukuran berulang, hasil pengukuran menjadi lebih akurat dan ketidakpastian pengukuran dapat ditentukan secara kuantitatif.
